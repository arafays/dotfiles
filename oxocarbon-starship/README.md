# Oxocarbon Starship Prompt

## Overview
The Oxocarbon Starship project provides a beautiful and customizable command line prompt based on the Oxocarbon theme, which is inspired by the IBM Carbon palette. This project includes configurations for the Starship prompt, examples of minimal and full setups, and installation instructions.

## Installation
To install the Starship prompt with the Oxocarbon theme, run the following command in your terminal:

```bash
curl -sS https://starship.rs/install.sh | sh
```

After installation, you need to configure your shell to use Starship. Follow the instructions specific to your shell in the [Starship documentation](https://starship.rs/).

## Configuration
The configuration for the Starship prompt can be found in the `config/starship.toml` file. This file allows you to customize the appearance and behavior of your prompt. 

### Example Configurations
- **Minimal Configuration**: Check out `examples/minimal.toml` for a simple setup.
- **Full Configuration**: For a comprehensive setup, refer to `examples/full.toml`.

## Usage
Once installed and configured, you can start using the Oxocarbon Starship prompt in your terminal. The prompt will display various segments such as the current directory, git status, and more, styled according to the Oxocarbon theme.
