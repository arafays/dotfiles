# spicetify Integration with Caelestia

## Configuration Overview
Spicetify is configured in Caelestia with a custom "caelestia" theme and marketplace integration.

## Installation Command
According to Caelestia docs:
```
spicetify config current_theme caelestia color_scheme caelestia custom_apps marketplace
spicetify apply
```

## Features
- **Theme**: Custom "caelestia" theme
- **Color Scheme**: "caelestia" color scheme
- **Marketplace**: Enabled for additional extensions
- **Custom Apps**: Support for custom applications

## Integration Notes
The spicetify configuration assumes the caelestia theme files exist in the Spicetify themes directory. The theme likely uses colors that match the overall Caelestia color palette.

## Repository Structure
Located in `spicetify/Themes/caelestia/` directory, though the color.ini file appears to be missing based on subagent analysis.
